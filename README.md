# M3U to MP3
This program is a C# version of the python M3U to MP3 converter
---
# How to use
- Compile the code in Visual Studio or go to the releases tab for the .exe
- Select the file, folder where the songs are contained and the folder to save the playlist to. (More explained in "How to ensure M3U file is in the right place")
---
# How to ensure M3U file is in the right place
- Right click on the M3U file and open with notepad
- Ensure that the directory it is pointing to is reachable.
- For example:
  - "../song/song.mp3"
  - You can ignore ../ in the program and instead just put all the song folders and M3U file in one path!
  - If you are unsure an error log file will be made showing which songs the program could not discover.
